2. Почему Python + MySQL + tkinter?
Python — простой, популярный, отличные библиотеки для работы с БД

MySQL — бесплатная, надёжная, широко используется в индустрии

tkinter — встроен в Python, не требует доп. установок

Альтернативы: PostgreSQL + Java (Swing/JavaFX), C# + MS SQL Server (WinForms/WPF), SQLite + Python (без установки сервера)

3. ER-диаграмма
Что это? Графическое представление структуры БД: сущности (таблицы), их атрибуты (поля) и связи между ними.

Зачем? Для проектирования БД «на бумаге» до реализации, согласования с заказчиком, документации.

4. Типы связей между таблицами
Тип	Пример (Пиццерия)
1:1 (один к одному)	Пицца → Пицца_характеристики
1:M (один ко многим)	Категория → Пиццы (одна категория содержит много пицц)
M:M (многие ко многим)	Заказы → Пиццы (через таблицу Заказ_Пиццы)
5. Git: commit vs push
Команда	Что делает
git commit	Сохраняет изменения в локальном репозитории
git push	Отправляет локальные коммиты в удалённый репозиторий (GitHub)
6. Проверка установки Python и tkinter
cmd
python --version          # Python 3.13.x
python -m tkinter         # открывается тестовое окно
7. Меры безопасности за компьютером
Не хранить пароли в коде / Git

Использовать .gitignore

Скачивать ПО только с официальных сайтов

Регулярно сохранять работу

Не подключать чужие USB-накопители без проверки антивирусом

8. Что означает 127.0.0.1?
Локальный хост (localhost) — специальный IP-адрес, означающий «этот же компьютер». Используется для подключения к сервисам на своей машине.

9. Как добавить файл в Git (последовательность)
cmd
git add файл          # 1. Добавить в отслеживание
git commit -m "сообщение"   # 2. Зафиксировать локально
git push origin main        # 3. Отправить на GitHub
